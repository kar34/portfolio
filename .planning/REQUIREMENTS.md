# Kenny Raymond Portfolio — Requirements

## Project Goal
Build a pixel-perfect replica of kenny-raymond.com as a clean, static Astro + Tailwind CSS site.

## Stack
- Framework: Astro (static output, no SSR)
- Styling: Tailwind CSS
- Smooth scroll: Lenis
- Font: Poppins (400, 500, 600) via Google Fonts
- Deploy target: Vercel

## Requirements

### REQ-001: Astro project initialized in `portfolio/` subdirectory
### REQ-002: Tailwind CSS configured with Poppins font and black/white color scheme
### REQ-003: Lenis smooth scroll installed and initialized globally
### REQ-004: Base layout component with nav (logo left, Work/About links right, hamburger on mobile) and footer (name + 3 social links with arrow icons)
### REQ-005: Global styles — black background, white text, Poppins, no scrollbar
### REQ-006: Favicon configured (Assets/favicon.png)
### REQ-007: All assets copied from Assets/ into portfolio/public/

### REQ-008: Home page — hero (name, title, location, portrait, email, social icons)
### REQ-009: Home page — About section with bio snippet
### REQ-010: Home page — WORK section heading + 3 project cards (LinkedIn Video, Reactions, Nearby Friends)

### REQ-011: Work page — same 3 project cards

### REQ-012: About page — portrait (b&w), "About Me" heading, full 5-paragraph bio

### REQ-013: LinkedIn Video project page — title, hero image, Overview section, GIF, Impact section, two mockup images side-by-side, "← More Projects" link
### REQ-014: LinkedIn Reactions project page — title, hero image, Overview section, before/after image, Impact section, single phone mockup, "← More Projects" link
### REQ-015: Nearby Friends project page — title, hero image, Overview section, Problems (3 items), Design Improvements section, Impact section, bottom image, "← More Projects" link

### REQ-016: Pixel-perfect match to Mocks/ screenshots (desktop and mobile)

### REQ-018: No CSS animations or scroll-triggered transitions
